# 大纲预览 — 数字便利贴视图

## 金字塔原理验证

**中心论点**：古代黑陶的呈色工艺可按物理化学本质归纳为四大类型，需要层级化科技方法体系加以判别，同时注意埋藏干扰与工艺多解性挑战。

**逻辑流向**：
背景铺垫（中西方差异）→ 分类体系（四大类型）→ 方法体系（三层级）→ 挑战（误判与矛盾）→ 展望与总结（新技术 + 跨文化比较）
