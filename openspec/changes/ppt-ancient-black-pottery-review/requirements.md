# Requirements: 古代黑陶烧制工艺科技考古综述 — 组会汇报 PPT

## Target Audience

- **导师 + 同方向同学**（考古/文物学方向研究生）
- 熟悉基本术语（XRD、SEM、拉曼光谱等不需要额外解释）
- 对黑陶工艺细节可能不深入了解，需要清晰的分类框架展示

## Presentation Purpose

- **汇报学习成果**：展示对古代黑陶烧制工艺领域的全面梳理和理解深度
- 体现系统性思维：从分类到方法到挑战到展望的完整逻辑链
- 展示文献综述能力：中外文献的整合能力

## Key Messages (3-5 核心观点)

1. 古代黑陶的四大工艺分类（还原焙烧/渗碳/三段式烧成/抛光）各有不同的呈色物理化学本质
2. 科技判别方法已形成从无损到微区定量的层级化体系
3. 埋藏环境干扰和工艺多解性是当前面临的核心挑战
4. 同步辐射微区技术等新方法为原位无损判别开辟新途径
5. 中西方黑陶工艺的跨文化比较揭示人类材料技术发展的共性

## Tone & Style

- **数据驱动型**：突出表格对比、化学式、分析方法的技术细节
- 学术严谨但清晰易读
- 使用 Minimal Clean 风格（`styles/minimal.yaml`）
- 信息密度适中偏高（听众为专业方向研究生）

## Content Structure (按论文章节)

1. **封面** — 标题、作者、院系、课程信息
2. **目录/提纲** — 汇报结构概览
3. **引言/背景** — 黑陶定义、分布、中西方差异、研究意义
4. **分类体系**（重点章节）:
   - 四大分类总览（对比图/表）
   - 还原焙烧黑陶（机理 + 化学式 + 判别）
   - 渗碳黑陶（机理 + 温度窗口 + 判别）
   - 三段式烧成（三阶段流程 + 选择性致色机制）
   - 抛光黑陶（结构致黑原理 + 案例）
5. **研究方法**:
   - 三层级方法体系（无损→微损→微区）
   - 各方法优劣对比表
6. **误判与挑战**:
   - 埋藏环境干扰
   - 工艺多解性难题
   - 有损 vs 无损矛盾
7. **未来展望** — SR-μXRD, XANES, Micro-CT, 实验考古
8. **结语** — 中西方对比总结
9. **结尾页** — 感谢聆听

## Page Range

- 目标: 15-18 页
- 不华丽但详细，覆盖论文所有核心内容

## Constraints

- 化学式需要准确渲染（Fe₂O₃, Fe₃O₄, FeO 等）
- 需要呈现论文中的表格数据（表 2-1, 表 3-1, 表 3-2）
- SVG 格式输出（1280x720）
