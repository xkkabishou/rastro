# 黑陶分类体系 — 关键素材提取

> 来源：main.pdf 第 2 章"黑陶的分类体系与科学判别"（pages 3-7）

---

## 总览：四大工艺类型分类树

以呈色物质的物理化学本质为分类主干，古代黑陶烧制工艺归纳为四大类型（图 2-1、表 2-1）。实际考古遗物中两种甚至三种工艺叠加并不罕见（如陶寺遗址龙山文化黑陶兼具陶衣和渗碳特征），但按主导呈色因素仍可归入相应类别。

| 工艺类型 | 呈色物质 | 典型代表 | 核心判别手段 |
|--------|--------|--------|----------|
| **还原焙烧**（铁致黑） | Fe₃O₄/FeO（磁铁矿/方铁矿） | 灰陶深色端、仰韶深灰陶 | XRD 磁铁矿峰、复烧变红实验 |
| **渗碳**（碳致黑） | 游离碳（炭黑） | 龙山蛋壳黑陶、大溪内黑陶 | Raman D/G 峰、EDS 碳梯度线扫描 |
| **三段式烧成**（晶相致黑） | 磁铁矿/赫赛石微晶涂层 | 雅典黑光陶、坎帕尼亚黑光陶 | SEM-BSE 涂层/胎体分界、SR-μXRD |
| **抛光**（结构致黑） | 矿物定向排列 | 百济研磨黑陶、巴里科特黑衣陶 | PLM 偏光消光、EBSD 织构分析 |

## 2.1 还原焙烧黑陶（铁致黑）

### 呈色机理

- 最基础的黑陶呈色途径
- 缺氧窑炉气氛中，赤铁矿（Fe₂O₃）被还原为磁铁矿（Fe₃O₄）或方铁矿（FeO）
- 化学反应链：**Fe₂O₃ → Fe₃O₄ → FeO**
- 陶器从红色转变为灰色乃至深灰黑色
- 颜色深浅取决于：还原程度 + 胎体含铁量
- 当 Fe₂O₃ 含量超过 5% 且窑内氧分压足够低时，胎体可呈接近黑色的深灰

### 返红机制

- 冷却时需保持隔氧，否则磁铁矿会重新氧化为赤铁矿而"返红"

### 案例数据

#### Nodari 等

- 研究对象：意大利帕达纳地区伊特鲁里亚夹心陶
- 发现：黑色外壳与红色内芯之间存在清晰的 **Fe²⁺/Fe³⁺ 梯度**
- 意义：证实了还原—氧化气氛的分层控制

#### Maritan 等

- 借助控制实验，系统考察有机质含量对烧成气氛的影响
- 发现：原料中的有机质在升温过程中消耗氧气、营造局部还原微环境，是低温阶段胎体变黑的重要辅助因素

#### Lima 等

- 运用穆斯堡尔谱学测定不同烧成温度下 Fe²⁺ 以及 Fe³⁺ 的相对丰度
- 注意：指出铁价态与烧成条件之间并非简单的一一对应，具体样品的分析中应审慎解读

#### Xia 等

- 研究对象：永城地区龙山文化陶器
- 方法：XRF 以及 XRD 分析
- 结论：该地区深灰色陶器中磁铁矿的存在与较强的还原烧成条件相一致

### 科技判别特征（三项指标）

1. XRD 谱中出现**磁铁矿或方铁矿的特征衍射峰**
2. 复烧实验中在氧化气氛下重新加热后**颜色恢复为红色**
3. 热重分析的升温过程中**不出现由碳燃烧引起的失重**
4. 以上三项指标可与渗碳黑陶形成有效区分

## 2.2 渗碳黑陶（碳致黑）

### 呈色机理

- 中国新石器时代黑陶最主要的呈色工艺
- 基本原理：陶器烧成后期密闭窑炉、减少进气量，使有机燃料在缺氧条件下不完全燃烧，产生大量微小的碳黑颗粒

### 碳沉积两途径

1. **物理充填**：碳黑颗粒对陶器表面开放气孔的物理充填
2. **化学沉积**（歧化反应）：一氧化碳借助歧化反应 **2CO → CO₂ + C**，在较低温度下析出固态碳

### 温度窗口

- **刘鹏的系统实验**表明：**630 C 以下渗碳效果最佳**
  - 此时粘土矿物尚保留层间水以及巨大的比表面积
  - 对芳香烃挥发物具有强烈的化学吸附能力
  - 超过此温度后，伊利石脱去结构水，吸附能力骤降，渗碳效率显著下降

### 渗碳梯度

- 渗碳程度通常由表面向胎体内部逐渐递减

### 李文杰四种渗碳模式

- 研究对象：大溪文化黑陶
- 方法：外观观察 + 仿烧实验
- 复原了四种渗碳模式：
  1. **全身皆黑**
  2. **外红内黑**
  3. **上红下黑**
  4. **竖条黑道**
- 提出了**窑内渗碳**以及**窑外渗碳**两种操作方式

### 其他关键案例

#### 沈建兴等

- 研究对象：龙山黑陶
- 方法：显微结构 + 化学成分分析
- 发现：胎体中存在大量气孔并且被�ite质材料部分填充
- 结论：虽然胎体含铁量比较高，但**渗碳工艺才是黑陶呈色的主要缘由**

#### 朱铁权等

- 方法：XRD、拉曼光谱、扫描电镜
- 研究对象：淮河流域双墩遗址黑陶
- 结果：主要显色物质是**炭黑**，并且胎体内大量的植物残体可能与渗碳工艺存在关联

#### 潘坤容等

- 发现：六安双龙墓群出土的**西汉黑陶**表面涂有炭黑与生漆的混合物
- 意义：说明渗碳工艺在新石器时代之后仍以不同形式延续

### 拉曼 D/G 峰数据（科技判别核心特征）

- **D 峰**：约 **1350 cm⁻¹**
- **G 峰**：约 **1580 cm⁻¹**
- D/G 峰的出现可明确指示游离碳的存在
- **EDS 线扫描**：显示碳含量从表层向内部呈**明显递减趋势**
- **SEM 形貌**：可观察到纳米碳颗粒呈**珊瑚状或树枝状集合体**充填于气孔中

## 2.3 三段式烧成与黑光陶（晶相致黑）

### 工艺概述

- 地中海地区古典时期的黑绘陶以及黑光陶选用的涂层致色技术
- 与中国黑陶依赖碳沉积截然不同
- 工匠先在成形的陶坯表面涂刷一层经过精细淘洗的**富铁细泥浆**（粒径一般小于 **2 μm**）
- 然后进行"**氧化—还原—再氧化**"三段式烧成

### 三阶段温度/气氛参数（图 2-2）

#### 第 I 阶段：氧化阶段

- **温度**：升温至 ~800 C
- **气氛**：有氧开放气氛
- **涂层**：涂层（<2 μm 泥浆）中 Fe₂O₃ 赤铁矿
- **胎体**：粗颗粒陶土中 Fe₂O₃ 赤铁矿

#### 第 II 阶段：还原阶段

- **温度**：850-950 C
- **气氛**：封闭窑室 + 湿木投入
- **涂层**：Fe₂O₃ → Fe₃O₄，开始烧结玻化
- **胎体**：Fe₂O₃ → Fe₃O₄

#### 第 III 阶段：再氧化阶段

- **温度**：降至 ~800 C
- **气氛**：重新开放进气口
- **涂层**：Fe₃O₄ 保持 → **致密玻化，氧无法渗入**（黑色保持）
- **胎体**：Fe₃O₄ → Fe₂O₃（气孔丰富，迅速再氧化，返红）

### 选择性致色的关键

- 涂层富含 **K₂O、CaO** 等助熔剂
- 粒径 < 2 μm → 低温即可烧结
- 封闭气孔阻氧 → 涂层保持黑色
- 胎体粗颗粒结构疏松 → 氧气渗入 → 重新氧化为红色
- 形成**红色胎体配以黑色涂层**的经典外观

### 案例数据

#### Cianchetta 等

- 方法：微区拉曼光谱 + X 射线衍射
- 研究对象：古希腊黑光陶仿制品以及真品的对比研究
- 结论：涂层中的黑色相主要是**磁铁矿**以及**赫赛石**（hercynite, **FeAl₂O₄**）微晶

#### Solard 等

- 方法：微区二维 X 射线衍射（**μ-XRD₂**）+ SEM-EDS 分析
- 研究对象：小亚细亚出土的公元 4 世纪雅典化黑光陶
- 发现：涂层富含 **K₂O** 以及 **CaO** 等助熔剂
- 胎体与涂层在**背散射电子图像中界限分明**

#### Laciak 等

- 研究对象：中欧史前黑涂层陶器
- 结论：涂层的**高铁含量**以及**细颗粒质地**是形成类似效果的共性条件

### 科技判别核心特征

1. **SEM-BSE 图像**中涂层与胎体存在清晰的灰度分界
2. 涂层中 **Fe、K 含量显著高于胎体**
3. **SR-μXRD** 可检测到涂层中的磁铁矿或赫赛石微晶
4. 涂层由于已充分烧结玻化，在**复烧实验中不会重新变红**

## 2.4 抛光黑陶（结构致黑）

### 结构致黑原理

- 黑色外观**不依赖碳沉积或铁物相的改变**
- 通过器表的**精细磨光**使层状硅酸盐矿物定向排列
- 消除了可见光的漫反射，从而呈现出**深色光泽外观**
- 本质是"**结构致黑**"——矿物颗粒物理排列改变光学性质

### 案例数据

#### Kim 等

- 研究对象：朝鲜半岛百济汉城期的研磨黑陶
- 方法：SEM + 矿物分析
- 发现：百济黑陶的黑色表层**并未检测到额外的碳沉积或涂层**
- 结论：呈色完全归因于**表面矿物颗粒的物理排列**
- 充分验证了"结构致黑"的机理

#### Blackmore 等

- 将百济黑陶的生产技术与国家形成过程联系起来
- 指出研磨抛光作为一种**费工费时的表面处理工艺**
- 具有一定的**社会等级标识功能**

#### Uhm 等

- 方法：放射性核分析 + 磁性测量
- 研究对象：百济汉城期黑陶
- 系统表征了化学组成以及磁学特征
- 为**还原气氛的判别**提供了补充数据

#### Maritan 等

- 研究对象：巴基斯坦西北部巴里科特遗址出土的黑衣陶器
- 方法：详细的矿物学以及化学分析
- 发现：黑色衣层在**还原气氛下烧制而成**
- 衣层与胎体的化学组成差异不大
- 关键特征：衣层中的**层状矿物呈明显的定向排列**

#### Choi 等

- 研究对象：百济泗沘期陶器
- 方法：XRD、SEM + 化学分析
- 发现：抛光处理导致**表面粗糙度大幅降低、光泽度明显提升**

#### Moon 等

- 方法：XRD 分析百济黑陶中粘土矿物的结构变化
- 将粘土矿物结构变化作为**烧成参数的指示器**

### 科技判别核心特征

1. **PLM 偏光显微镜**下观察到表面矿物的**同步消光现象**（矿物定向排列的证据）
2. **EBSD**（电子背散射衍射）可表征层状矿物的**定向排列特征**（已被应用于考古陶瓷研究）
3. **AFM**（原子力显微镜）测量显示表面粗糙度**显著低于未抛光区域**
4. **EDS + 拉曼分析**中**不出现游离碳或磁铁矿信号**（排除渗碳和还原机制）

## 交叉参考：工艺叠加现象

- 实际考古遗物中两种甚至三种工艺叠加并不罕见
- 案例：陶寺遗址龙山文化黑陶兼具**陶衣和渗碳**特征
- 但按主导呈色因素仍可归入相应类别
- 呈色机理的差异直接影响了陶器的物理性能：
  - 渗碳黑陶：碳颗粒填充气孔 → **致密度提高**
  - 三段式黑光陶：表面玻璃化程度高、耐磨性突出
  - 抛光黑陶：兼具良好的**防渗性能**
