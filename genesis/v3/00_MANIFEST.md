# Genesis v3 - 版本清单

**创建日期**: 2026-03-18
**状态**: Active
**前序版本**: v2

## 版本目标
增加轻量级翻译功能：划词翻译、文献标题翻译缓存，以及独立的翻译 API 配置入口。

## 主要变更
- 新增划词翻译功能（选中文字后浮窗支持"翻译"选项）
- 新增选项浮窗毛玻璃重设计（与笔记风格一致）
- 新增 Zotero 文献标题中文翻译（hover 显示，带缓存）
- 新增 hover tooltip 毛玻璃风格
- 新增设置页面独立翻译 API 配置 Tab
